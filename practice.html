Random Exercises

These are unpolished Javascript functions that I have done as practice. Why not put them all somewhere?
Most of the code behind these functions can be found here.

Dollars and Cents

Given an amount of dollars and cents as a number, it will tell you the most efficient way to break down that dollar amount.

Fibonacci Generator

Generate a Fibonacci Sequence of your own! Tell it the first two terms and how many terms you'd like to see (less than 100)!








Quadratic Formula

Given a parabola written in expanded ax2+bx+c=0 form, input the a, b, and c values to get the roots! Supports Imaginary number answers!








Random Number Generation

Checks how many tries it takes hit 1 out of the number you input. It is expected to give different answers for the same number, as each time is different RNG.

Unique character check

Finds any unique characters (only 1 occurrence) from an input and then outputs them in the order they appeared. Also makes letters lowercase.

Example: "Hello how are you" becomes "helo waryu"


Prime Checker

Checks if a number is prime. Only works for smaller numbers, less than the 32 bit integer overflow.

Simple Factorial

Given an positive integer, find the factorial. Doesn't work for large inputs

Simple Add Commas to Number

Given a number, add commas to it! Works with decimals and negatives as well!
